## Generate Icons (Required First Step)

Before loading the extension, you need to generate the icon files:

1. Open `icons/generate-icons.html` in your Chrome browser
2. The icons will be automatically generated on the page
3. Right-click on each icon (16x16, 48x48, 128x128)
4. Select "Save image as..."
5. Save each as:
   - `icon16.png` (16x16 icon)
   - `icon48.png` (48x48 icon)
   - `icon128.png` (128x128 icon)
6. Save all three PNG files in the `icons/` folder

## Load Extension in Chrome

1. Open Chrome and go to `chrome://extensions/`
2. Enable "Developer mode" (toggle in top right)
3. Click "Load unpacked"
4. Select the `yandex-maps-chrome-extension` folder
5. Done! The extension is now installed

## First Use

1. Go to [https://yandex.ru/maps](https://yandex.ru/maps)
2. Search for businesses (e.g., "Restaurants Moscow")
3. Click the extension icon in your Chrome toolbar
4. Set your target (default: 10 leads)
5. Click "Start Scraping"
6. Watch as qualified leads are found!
7. Export as CSV or JSON when done

## Troubleshooting

**Icons missing error?**
- Make sure you generated and saved all 3 icon files
- Check that they're named exactly: icon16.png, icon48.png, icon128.png
- Verify they're in the `icons/` folder

**Extension not loading?**
- Make sure all files are in the correct folder structure
- Check Chrome console for error messages
- Try reloading the extension
